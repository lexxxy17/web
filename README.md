# КроссМаркет — демо-магазин кроссовок (HTML/CSS/JS)

Простой статический сайт из 5 страниц на чистом HTML, CSS и Vanilla JS:

- Главная `index.html`
- Магазин `shop.html`
- О нас `about.html`
- Контакты (с валидацией) `contacts.html`
- Корзина `cart.html`

Общий шапка-навигация присутствует на каждой странице. Корзина и счётчик товаров работают в `localStorage`.

## Запуск локально

В корне проекта запустите простой сервер (вариант на Python):

```bash
cd /workspaces/web
python3 -m http.server 8000
```

Откройте в браузере: `http://localhost:8000/index.html`

## Структура

```
assets/
	css/
		styles.css      # Общие стили и оформление
	js/
		main.js         # Навигация, корзина, валидация формы контактов
index.html          # Главная
shop.html           # Каталог товаров
about.html          # О компании
contacts.html       # Контакты с формой и валидацией
cart.html           # Корзина (localStorage)
```

## Функции

- Навигация: активный пункт подсвечивается согласно текущей странице.
- Магазин: добавление товаров в корзину (демо-набор карт).
- Корзина: изменение количества, удаление, очистка, подсчёт суммы.
- Контакты: валидация имени, email, телефона (опционально) и сообщения, показ успеха, запись заявки в `localStorage` (демо).

## Примечания

- Все данные демо-режима хранятся в `localStorage` и очищаются вручную.
- Стили — тёмная тема, адаптивная сетка.
